## Requirements

- Node (LTS)
- Yarn
- GIT

## Installation

- `$ git clone https://github.com/emmanuelcestoni/WindowEstimatingFrontEnd.git`
- Go to the project root directory and install dependencies:

### Local development

```
yarn install
yarn run dev
```

### Production deployment

```
yarn install
yarn run build
yarn run start
```
